Welcome to Badge 101

Files

JSON files

Issuer
Who issued the badge: name, description, image, url, email, revocationList (for signed badges)
Badge Class
The kind of badge: name, description, image, tags, criteria (url), issuer (json url), alignment (list w name, description, url)
Assertion
Awarded Badge: uid, recipient email (can be hashed), image (with this very own info baked in it), evidence (url), issuedOn, expires, badge (json url), verify (hosted: url to self! or signed: url to public key)
Revocation List
For signed assertions only: hash of the signed keys of revoked assertions with explanation
